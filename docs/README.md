# Папка для шаблонов документов

В этой папке размещаются файлы, на которые ссылается раздел `templates.html`
(«Пакет документов для инвестора»). На странице ссылки уже проставлены —
достаточно положить файлы сюда с указанными именами, и кнопки скачивания заработают.

## Перечень шаблонов

| Файл (pdf + docx)                   | Назначение                                              |
|-------------------------------------|---------------------------------------------------------|
| `teaser_investor`                   | Инвестиционный тизер (1–2 страницы)                     |
| `investor_note`                     | Расширенная инвестиционная заметка                      |
| `preliminary_contract_colocation`   | Предварительный договор аренды стойко-мест              |
| `app1_parameters`                   | Приложение № 1: коммерческие и технические параметры    |
| `project_passport`                  | Паспорт проекта для размещения на площадках             |
| `letter_of_intent`                  | Письмо о намерениях (якорный клиент)                    |
| `confidentiality_agreement`         | Соглашение о конфиденциальности (NDA)                   |
| `tech_concept_brief`                | Техническая концепция: состав исходных данных           |
| `platform_project_card`             | Карточка проекта для торговых площадок                  |
| `deal_outline_onepager`             | Контур сделки (1 страница)                              |

Каждый документ нужен в двух форматах: `.pdf` (для просмотра) и `.docx` (для редактирования инвестором).

## Пример

```
docs/
├── README.md                              ← этот файл
├── teaser_investor.pdf
├── teaser_investor.docx
├── investor_note.pdf
├── investor_note.docx
└── ...
```

Пока документы не загружены, ссылки в `templates.html` будут вести на 404 GitHub Pages —
это нормально для этапа подготовки сайта к публикации.
